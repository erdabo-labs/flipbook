# Flipbook brand assets

## Primary logo (the missing piece — now included)
- `svg/flipbook-logo-horizontal.svg` — **primary lockup**: coin mark + FLIPBOOK (FLIP ink, BOOK emerald)
- `svg/flipbook-logo-horizontal-reversed.svg` — for dark/emerald backgrounds (white + mint)
- `svg/flipbook-logo-stacked.svg` — square/narrow contexts
- `png/flipbook-logo-horizontal.png` · `…-reversed.png` · `…-stacked.png` — 3× raster of each

> SVGs are vector + editable. The wordmark uses **Hanken Grotesk 800** (−3.5% tracking);
> the font is referenced by name, so open in a browser/Figma/Illustrator that has it.
> The PNGs are pre-rendered with the font baked in for slides, email, etc.
